SEMÂNTICA HTML

!DOCTYPE HTML > declaração define que este documento é um documento HTML5

HTML > elemento é o elemento raiz de uma página HTML

HEAD> elemento contém meta informações sobre a página HTML

TITLE > elemento especifica um título para a página HTML (que é mostrado na barra de título do navegador ou na guia da página)

BODY > elemento define o corpo do documento e é um contêiner para todo o conteúdo visível, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.

H1 > elemento define um grande título

P > elemento define um parágrafo

Elementos de conteúdo inline

A> hyperlink, ligando a outro recurso

EM> ênfase do conteúdo

STRONG> importância de um trecho do texto, não altera sentido da frase

SMALL> font pequena, aviso legal, não essencial ao para entendimento

S> sublinhado

CITE> título de uma obra

Q > curta citação

DFN > termo cuja definição está contida em seu conteúdo ancestral mais próximo

ABBR > abreviatura ou acrónimo, eventualmente, com seu significado

TIME > valor de data e hora

CODE > representa um codificação

VAR > representa uma variável

SAMP > representa uma saída de um programa de computador

KBD > representa uma entrada do usuário/texto,comando de voz transcritos

SUB > subscrito

SUP > sobrescrito

B > negrito

SPAN> texto sem significado semântico

BR > pula linha

HR > Insere uma linha divisória

Tags para separação de conteúdo

SECTION > define uma seção em um documento.

NAV > define um conjunto de links de navegação. o elemento é destinado apenas para os principais blocos de links de navegação.

ARTICLE > tag especifica conteúdo independente e autocontido. Postagem no fórum,Postagem do blog, Notícia.
um ou mais elementos de título (H1 - H6), logotipo ou ícone,informações de autoria
Nota: Você pode ter vários HEADER elementos em um documento HTML. No entanto, HEADER não pode ser colocado dentro de um FOOTER, ADDRESS ou outro HEADER elemento.


ASIDE > elemento define algum conteúdo além daquele em que é colocado (como uma barra lateral).


A1- H6 > As tags H1-H6 são usadas para definir títulos HTML.
H1 define o título mais importante. H6 define o título menos importante. Nota: Use apenas um H1 por página - isso deve representar o título/assunto principal para a página inteira. Além disso, não pule os níveis de título - comece com H1, depois use H2, e assim por diante.


HGROUP > tag é usada para cercar um título e um ou mais P elementos.
O título dentro do HGROUP elemento pode ser qualquer um dos H1 títulos H6.
Nota: o HGROUP elemento não é renderizado como algo especial em um navegador. No entanto, você pode usar CSS para estilizar o HGROUP elemento e seu conteúdo.


HEADER > elemento representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação.

elemento define um rodapé para um documento ou seção.
normalmente contém:informações de autoria ,Informações sobre direitos autorais, informações de contato, Mapa do site, voltar aos links principais, documentos relacionados.

ADDRESS > tag define as informações de contato do autor/proprietário de um documento ou artigo.
As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, conta de mídia social, etc.


MAIN >tag especifica o conteúdo principal de um documento.
O conteúdo dentro do MAIN elemento deve ser exclusivo para o documento. Ele não deve conter nenhum conteúdo que seja repetido em documentos, como barras laterais, links de navegação, informações de direitos autorais, logotipos de sites e formulários de pesquisa.
Nota: Não deve haver mais de um MAIN elemento em um documento. O MAIN elemento NÃO deve ser descendente de um elemento ARTICLE, ASIDE, FOOTER, HEADER ou NAV.
