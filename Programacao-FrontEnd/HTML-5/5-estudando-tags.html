ARTICLE

Espaço na página para algum contéudo independente, como texto, formulário

ASIDE

contéudo relacionado com a página

COMMAND

Botão, radiobutton ou checkbox para formulários

DETAILS

Detalhamento de uma parte da página

SUMMARY

Capitular ou sumário dentro da tag details

FIGURE

Legenda em uma imagem ou vídeo

FIGCAPTION

Legenda uma figura em uma sessão

FOOTER

Rodapé de um documento ou sessão,podendo incluir:data, autor, contatos e direitos autorais

HEADER

Usado para introduzir um documento ou sessão e pode conter o elemento NAV para inclusão de lins para navegação

HGROUP

Cria títulos e subtitulos a parir das tags H1 a H6,em que o maior é o título e demais subtítulos

MARK

Marcador de textos

METER

Medição, usado quando existe valor maximo e minimo conhecidos

NAV

Usado para um sessão de navegação por links, menu

PROGRESS

Progresso de uma tarefa

SECTION

Sessão de u documento, como capitulos, cabeçalhos, rodapés ou outras sessões

TIME

Definição de tempo ou data

WBR

Quebra de palavra, para criar quebra de linha
